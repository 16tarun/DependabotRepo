# DependabotRepo

Starter manifest for the **Dependabot** dependency-scanning GitHub Actions lab.

This repository contains a minimal npm project whose `package.json` declares a
few dependencies. Copy these files into your own repository, then add the
Dependabot configuration at `.github/dependabot.yml` as described in the lab
instructions.

```
DependabotRepo/
├── package.json
└── index.js
```

> Note: This repo intentionally does **not** include `.github/dependabot.yml` —
> creating it is part of the lab task.
